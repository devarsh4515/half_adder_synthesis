# Half Adder using Verilog HDL

## Overview
This repository contains the RTL implementation of a Half Adder using Verilog HDL along with synthesis using Yosys.

## Repository Contents
- RTL Verilog source code
- Testbench
- Yosys synthesis script
- Synthesized netlist
- Yosys DOT graph
- Synthesized circuit image
